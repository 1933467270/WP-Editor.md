== Description ==

WP Editor.MD是一个漂亮又实用的在线Markdown文档编辑器。

WP Editor.MD is a beautiful and practical Markdown document editor.

基于Editor.md构建对WordPress平台的支持。

Build support for the WordPress on Editor.md.

使用WordPress Jetpack 的Markdown模块来解析和保存内容。

Use the Markdown module of WordPress Jetpackto parse and save content.

详细介绍： https://iiong.com/wordpress-plugins-wp-editormd.html

QQ 群：246727127

== Installation ==

1. 上传 `WP-Editor.MD`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
