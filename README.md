# WP Editor.md


### Description

WP Editor.md是一个漂亮又实用的在线Markdown文档编辑器。

基于[Editor.md](https://github.com/pandao/editor.md)构建对WordPress平台的支持。

使用WordPress[Jetpack](http://jetpack.me/support/markdown/)的Markdown模块来解析和保存内容。
